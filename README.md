## Project Description 
A JavaFX-based application simulating a food truck ordering system, allowing customers to place orders and managers to track inventory and revenue.

## Technologies
- Java
- JavaFX
- OOP, GUI Design

## Features
- Customer interface: add/remove items, apply coupons, and update stock dynamically in real time
- Manager interface: restock inventory and track revenues and sales metrics
- Robust error handling: prevent out-of-stock purchases and ensure accurate order processing
